Codengine

Pure open-source PHP framework for small business

  Discover

If you are new to the PHP framework Codengine then the best way to see what it's all about is to read through our Documentation.

Any open-source project, especially framework requires great documentation. If you'd like to write your own guides and tutorials, feel free to get in touch with us at danielg6666@me.com.

  Features

Codengine is a MVC (Model-View-Controller) framework that was designed to fully support the HMVC architectural pattern. But there's more! We also provide you with pre-built base classes to help you be faster and more productive.

Codengine also gives you seo friendly routing capabilities. You can switch between navigation modes by an easy-to-use configuration file.

  Contribute
